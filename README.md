# 🌌 Stellar OS | Advanced Command Center

**Stellar OS** is a high-fidelity, sci-fi inspired command center interface designed for visualizing the solar system in real-time. It features procedural space generation, dynamic orbital mechanics, and an interactive telemetry system.

## 🚀 Key Features

* **Procedural Universe Engine:** Generates deep-space nebulae and thousands of stars dynamically using the HTML5 Canvas API.
* **Hyperspace Transition:** An immersive "Warp" effect that seamlessly transitions the user from the landing page to the main command interface.
* **Dynamic Orbital Mechanics:** Each planet follows calculated elliptical paths with real-time speed controls and time-dilation settings.
* **Telemetry & Intelligence Data:** An interactive glass-panel UI that provides detailed scientific data (Atmospheric composition, gravity, escape velocity, etc.) for every celestial body.
* **Command Interface:** * **360° Viewport Control:** Pan and zoom the system map with fluid inertial scrolling.
    * **Focus Tracking:** Click any planet or the Sun to center the viewport and pull up its classified intelligence report.
    * **Timeline Controls:** Pause or accelerate the orbital mechanics using the integrated time-dilation slider.

## 🛠 Tech Stack

* **Core:** HTML5, CSS3 (Modern Flexbox/Grid), and Vanilla JavaScript (ES6+).
* **Visuals:** HTML5 Canvas API (procedural generation), CSS Animations, and Glassmorphism design principles.
* **Interactivity:** Custom event listeners for drag-to-pan, scroll-to-zoom, and state-based UI transitions.
* **Typography:** Google Fonts (Rajdhani) for that authentic high-tech UI aesthetic.

## 🕹 Controls

* **Left Click:** Select a planet to pull up the intelligence report.
* **Drag:** Pan around the solar system map.
* **Scroll:** Zoom in to focus on specific planets or zoom out to view the entire sector.
* **Time Dilation:** Use the slider in the bottom control panel to speed up or slow down planetary rotation.
* **Pause/Resume:** Click the timeline button to freeze or resume orbital movement.

## 🎨 Design Philosophy

Stellar OS is built to simulate a high-tech tactical console. The interface utilizes high-contrast accents (`#38bdf8`), scanline overlays, and glass-blur effects to ensure maximum readability while maintaining a "Command Center" atmosphere. Every component is optimized for performance, ensuring the 2D orbital math and canvas rendering remain smooth at 60fps.
